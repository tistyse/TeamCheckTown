# TeamCheckTown

A Python-based team availability and status checking application. TeamCheckTown helps manage and track the availability of team members with a friendly interface and straightforward status updates.

## Features

- **Team Status Tracking**: Monitor the current availability status of team members
- **Easy Status Updates**: Quickly update team member availability
- **Friendly Interface**: User-friendly design for seamless interaction
- **Team Management**: Organize and manage team member information

## Files

- `Teamcheck.py` - Main application script containing core functionality
- `friendly.txt` - Configuration or reference file with team-related data
- `README.md` - Project documentation

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
```bash
git clone https://github.com/tistyse/TeamCheckTown.git
cd TeamCheckTown
```

2. No additional dependencies required for basic usage.

## Usage

Run the main application:
```bash
python Teamcheck.py
```

## Getting Started

The application provides a straightforward way to:
1. Check team member availability
2. Update status information
3. View team overview
